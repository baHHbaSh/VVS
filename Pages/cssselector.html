css селекторы

Чтобы редактировать только нужные элементы при помощи css, были придуманы селекторы

Селекторов есть 3 типа: по html тегу, по class и по id

Выглядят они следующим образом:

У нас есть элемент:

текст

p{color: black}
.par{color: black}
#first{color: black}

Абсолютно каждый указатель сделает цвет данного элемента чёрным

Каждый тип селектора имеет приоритетность:

Селектор по тегу имеет самый низкий приоритет

Селектор по классу имеет средний приоритет

Селектор по id имеет высокий приоритет

Самый высокий приоритет имеет стиль определённый в самом html элементе, пример приведён ниже
