Начало изучения HTML

Что такое HTML

html - декларативный язык разметки. Он не является языком программирования! По факту html представляет из себя только инструкции для браузера, описывая построение станицы

Структура html - теги. В html есть 3 типа тегов: Открывающие, закрывающие и "полноценные". Закрывающий тэг всегда повторяет открывающий, но имеет slash в своём синтаксисе
