Элемент <input> используется для создания интерактивных элементов управления для веб-форм, чтобы принимать данные от пользователя. В зависимости от устройства и пользовательского агента доступны различные типы входных данных и элементов управления.

То, как работает <input> значительно различается в зависимости от значения его атрибута type, поэтому различные типы описаны на отдельных справочных страницах. Если этот атрибут не указан, используется тип по умолчанию text.
