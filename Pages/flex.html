Свойство display

display - единственный вариант создание одностраничных сайтов (без js фреймворков), чаще всего используется 3 варианта display:

block

flex

none

display: block; - используется из-за того что его html сам устанавливает

display: none; - используется для того чтобы сделать элемент "невидимым" (а точнее уменьшает размеры до 0) т.е. элемента как-будто нет

display: flex; - самый лучший выбор из-за того что он позволяет удобно перемещать элементы по экрану.

У flex отображения есть несколько важных свойств

flex-direction имеет значения: column (в столбик), row (в строчку), column-reverse, row-reverse. Flex Выставляет компоненты по заданному направлению

justify-content имеет значения: center, flex-start, flex-end, space-around space-between, stretch

align-items имеет значения flex-start, flex-end, center, baseline

Все эти значения показываются в инспекторе google chrome (а возможно и других нормальных браузеров). Рядом со style display:flex; есть кнопка в которой все эти значения можно визуально редактировать.
