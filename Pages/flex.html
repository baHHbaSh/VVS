Свойство display

display - единственный вариант создание одностраничных сайтов (без js фреймворков), чаще всего используется 3 варианта display:

block

flex

none

display: block; - используется из-за того что его html сам устанавливает

display: none; - используется для того чтобы сделать элемент "невидимым" (а точнее уменьшает размеры до 0) т.е. элемента как-будто нет

display: flex; - самый лучший выбор из-за того что он позволяет спокойно перемещать элементы по экрану:
