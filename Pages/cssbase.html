CSS

Одна из самых сложных тем - css. Без css сайты бы выглядели как документы в блокноте.

Также css является очень сложной, на первый взгляд, темой. И действительно в css много свойств, некоторые даже могут конфликтовать между собой, а также нужно изучать css селекторы (которые в позже пригодятся в js)

Подключение .css файла к html производится в теге <head> при помощи тега link

<link rel="stylesheet" href="style.css">

Синтаксис css - ключ - значение. Обратимся к примеру из реальной жизни: ёлка - зелёная. Синтаксис css работает так - селектор{свойство: значение свойства; Другое свойство: значение свойства...}

С ёлкой, css будет выглядеть подобным образом: ёлка{цвет:зелёный;}

Остаётся только узнавать свойства и их значения, чтобы измнять эту "ёлку". А о том как найти эту ёлку будет на следущей странице

cssselector
